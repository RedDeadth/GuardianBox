## English

IoT smart locker management system with remote control, real-time synchronization and native Android app.

### Repositories

| Component | Stack | Link |
|---|---|---|
| Web Panel + Backend | Django, React | [GuardBoxing](https://github.com/RedDeadth/GuardBoxing) |
| Android App | Kotlin, MVVM, Firebase | [WasabiDefinitive](https://github.com/RedDeadth/WasabiDefinitive) |

### Architecture

- Web panel with Django (MVC) and React for administration and monitoring
- Android app with MVVM pattern and LiveData for reactive UI
- Firebase Realtime Database for instant synchronization between all components
- Arduino firmware in C++ for physical locker open/close control

## Español

Sistema IoT de gestión de casilleros inteligentes con control remoto, sincronización en tiempo real y aplicación Android nativa.

### Repositorios

| Componente | Stack | Link |
|---|---|---|
| Panel Web + Backend | Django, React | [GuardBoxing](https://github.com/RedDeadth/GuardBoxing) |
| App Android | Kotlin, MVVM, Firebase | [WasabiDefinitive](https://github.com/RedDeadth/WasabiDefinitive) |

### Arquitectura

- Panel web con Django (MVC) y React para administración y monitoreo
- App Android con patrón MVVM y LiveData para UI reactiva
- Firebase Realtime Database para sincronización instantánea entre todos los componentes
- Firmware Arduino en C++ para control físico de apertura y cierre de casilleros
